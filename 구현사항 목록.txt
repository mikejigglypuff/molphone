교사
뒤도는 애니메이션
움직이는 애니메이션

학생
핸드폰 꺼내고 전원 키는/전원 끄고 집어넣는 애니메이션, 이 때 줌 인/아웃도 이루어져야 함

핸드폰
각 앱을 누르면 잠시 까매지거나 하얘졌다가 앱이 실행되는 애니메이션
홈 버튼을 누르면 바탕화면으로 나가지는 애니메이션
인터넷 탭을 눌렀을 때 스크롤을 내려가며 탭을 보는 애니메이션
유튜브 스크롤을 쭉 내리는 애니메이션
화면을 눌렀을 때 영상이 재생되는 애니메이션
채팅이 계속 올라가는 애니메이션
자동사냥 애니메이션
육성게임의 스틱맨 애니메이션들

기능
꿀잼 게이지가 시간에 따라 감소하는 시스템 (이 시스템이 필요한지에 대한 고민 필요)
제한 시간내로 게이지를 다 채우지 못하면 게임 오버되는 시스템
교사의 시야에 플레이어의 핸드폰 하는 모습이 잡히면 게임 오버되는 시스템
핸드폰의 탭/홈/뒤로가기 버튼
각 앱을 누르면 해당 앱이 실행되는 기능
  1. 인터넷을 눌렀을 때 최소 하나 이상은 반짝이는 탭 포함하는 인터넷 창이 뜨도록 하는 기능
  2. 유튜브를 눌렀을 때 영상들이 계속 스크롤로 넘어가며 랜덤한 주기로 반짝이는 영상이 뜨도록 하는 기능
  3. 카톡을 눌렀을 때 메시지들이 계속 스크롤로 넘어가며 잠깐동안 반짝이는 것을 포함한 선택지들이 나오도록 하는 기능
  4. 추가 예정 
교사가 일정 시간동안 뒤돌도록 하며 앞을 보는 경우 시야를 몇가지 패턴 중 랜덤한 패턴으로 움직이는 기능
시작화면에서 전달된 난이도에 따른 꿀잼 게이지 감소/증가량, 교사의 시야 이동속도, 제한시간 등을 조절하는 시스템
